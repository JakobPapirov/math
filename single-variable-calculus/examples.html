Power rule

f ( x ) = 3 x 2 f ′ ( x ) = 2 · 3 x 2 - 1 f ′ ( x ) = 6 x

Sum rule

f ( x ) = 3 x 2 + 4 x 3 f ′ ( x ) = 2 · 3 x 2 - 1 + 3 · 4 x 3 - 1 f ′ ( x ) = 12 x 2 + 6 x

The differentiated between functions can be derived in an analogous way.
